# Demo Script — Orbital Nexus

## Setup (Before Demo)

- [ ] Backend running on `localhost:8000`
- [ ] Frontend running on `localhost:5173`
- [ ] Test `/health` endpoint returns OK
- [ ] Map loads with Greater Noida centered

## Demo Flow (3 minutes)

### 1. Opening (30s)

- Show the landing page — map centered on Greater Noida
- Explain: "This is Orbital Nexus — a multi-satellite data fusion dashboard."

### 2. First Query (60s)

- Type: **"What is the soil moisture and NDVI trend for Greater Noida this month?"**
- Show: Fused data cards appear in left sidebar
- Show: Map updates with relevant overlay
- Explain: "We're fusing weather, soil, and vegetation data from multiple satellites."

### 3. Second Query (60s)

- Type: **"Recommend the best crop for this region based on current conditions."**
- Show: AI-generated crop recommendation card
- Show: Guidance text with reasoning
- Explain: "The system cross-references soil health, weather patterns, and crop suitability."

### 4. Third Query (30s)

- Type: **"Show me flood risk for the next 7 days."**
- Show: Risk overlay on map + alert card
- Explain: "Multi-source fusion enables early warnings that single-satellite systems miss."

### 5. Closing (30s)

- Summarize: "Orbital Nexus turns raw satellite data into actionable intelligence — offline, instant, and accessible."
