# Orbital Nexus - Project Context

## What is Orbital Nexus?

- Micro-SaaS platform for guided farming using multi-satellite data fusion
- Built for Hyperspace Innovation Hackathon 2026 (36-hour timeline, Feb 7-8)
- Addresses Indian agriculture supply-demand imbalances through AI-driven crop recommendations
- Fuses public satellite data (Sentinel, Landsat, ISRO) with soil, weather, and mandi price datasets
- Provides farmers with crop recommendations, price predictions, and sustainability guidance
- Demo-first mindset: Visual impact, stability, and judge comprehension prioritized over algorithmic depth
- Ethical focus: Public data only, guidance language (not predictions), sustainability-driven
- Target deployment: orbitalnexus.online (Hostinger)
